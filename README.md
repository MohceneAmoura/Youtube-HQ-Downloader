# 📹 YouTube HQ Downloader

Une application Python avec interface Tkinter élégante pour télécharger des vidéos YouTube en haute qualité (jusqu'à 4K).

---

## 🚀 Fonctionnalités

- **Haute qualité** : Téléchargement en 360p, 480p, 720p, 1080p, 1440p ou 2160p (4K)
- **Interface moderne** : Thème sombre élégant avec Tkinter
- **Facile à utiliser** : Collez le lien, choisissez la qualité, téléchargez !
- **Fusion audio/vidéo** : Utilise FFmpeg pour combiner vidéo et audio en haute qualité

---

## 📦 Installation

### Prérequis

- Python 3.7+
- FFmpeg (pour les hautes qualités)

### 1. Cloner le repository

```bash
git clone https://github.com/VOTRE-PSEUDO/Youtube-HQ-Downloader.git
cd Youtube-HQ-Downloader
```

### 2. Installer les dépendances

```bash
pip install -r requirements.txt
```

### 3. Installer FFmpeg (optionnel mais recommandé)

Voir le fichier [`INSTALL_FFMPEG.md`](./INSTALL_FFMPEG.md) pour les instructions d'installation.

---

## 💻 Utilisation

1. Lancez l'application :
   ```bash
   python youtube_downloader.py
   ```

2. Collez le lien de la vidéo YouTube

3. Choisissez la qualité désirée

4. Sélectionnez le dossier de destination

5. Cliquez sur **"Télécharger"** !

---

## 🛠️ Technologies utilisées

- **[pytubefix](https://github.com/JuanBindez/pytubefix)** : Pour télécharger les vidéos YouTube
- **[Tkinter](https://docs.python.org/fr/3/library/tkinter.html)** : Pour l'interface utilisateur
- **[FFmpeg](https://ffmpeg.org/)** : Pour fusionner audio et vidéo
